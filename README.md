##Database structure
第一版构建了非常简易的数据库，有一些表格的构思还不够完善，随后会逐渐加入代码中。
目前有usr表，相册表，照片表，还有一些内容比如ai功能，角色分工，举报表。
对于管理员做简要说明：用户可以举报照片或者用户之类的，管理员收到举报之后可以选择删除系统中的照片，删除用户等。
更多功能如分享照片，系统通知等功能陆续实现
